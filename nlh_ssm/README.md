# nlh_ssm/

Core Python package: hyperbolic geometry, NL-H blocks, data loading, and evaluation metrics.

| Subdirectory | Role |
|--------------|------|
| `ops/` | Poincaré ball operations and NaN guards |
| `modules/` | NL-H block and hierarchy embedding |
| `models/` | Sequence mixer and selective state update |
| `data/` | CSV sliding-window loader with hierarchy metadata |
| `metrics/` | sMAPE, RMSSE, ACD, MAT |
| `utils/` | Shared metric primitives |

See [FILE_INDEX.md](../FILE_INDEX.md#nlh_ssm--core-library) for per-file comments.
